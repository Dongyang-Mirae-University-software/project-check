# 🔍 높은 정확도인데 감지가 안 되는 이유 (공통)

| 원인 | 영향도 | 설명 | 해결책 |

| ③ 작은 객체 | ⭐⭐⭐⭐ | 객체가 작거나 멀 때 감지 어려움 | 거리 조정 또는 이미지 크기 증가 |
| ④ 조명 변화 | ⭐⭐⭐⭐ | 학습 데이터보다 어둡거나 너무 밝은 환경 | 다양한 조명 조건 데이터 필요 |
| ⑤ 각도/포지션 | ⭐⭐⭐ | 특정 각도나 포지션에서 약함 | 다양한 각도 데이터 추가 |
