# 🔍 높은 정확도(mAP50 88.6%)인데 감지가 안 되는 이유

## 5가지 주요 원인 분석

| 원인 | 설명 | 영향도 | 해결책 |
|------|------|--------|--------|
| **Domain Shift** | Validation 데이터와 카메라 환경이 완전히 다름 | ⭐⭐⭐⭐⭐ | 추가 데이터 수집 또는 Fine-tuning |
| **신뢰도 임계값** | conf=0.5 기본값이 너무 높아서 낮은 신뢰도 감지 무시 | ⭐⭐⭐⭐⭐ | **conf=0.3으로 바로 조정 (가장 빠른 해결)** |
| **작은 객체** | Knife가 작거나 멀리 있으면 감지 어려움 | ⭐⭐⭐⭐ | 거리 조정 또는 이미지 크기 증가 |
| **조명 변화** | 학습 데이터보다 어둡거나 너무 밝은 환경 | ⭐⭐⭐⭐ | 다양한 조명 조건 데이터 추가 |
| **각도/포지션** | 학습하지 않은 특정 각도에서 약함 | ⭐⭐⭐ | 다양한 각도 데이터 추가 |

---

## 🎯 즉시 실행 순서

### 1️⃣ **신뢰도 조정 (5분)** ← 가장 먼저!
```python
# multi_detection.py에서:
# conf=0.5 → conf=0.3
results = model(frame, conf=0.3, verbose=False)
```

### 2️⃣ **조명 환경 테스트 (10분)**
- ✅ 밝은 실내 (일반적인 환경)
- ✅ 어두운 실내 (야간 조명만)
- ✅ 자연광 (창가 근처)

### 3️⃣ **거리 테스트 (5분)**
- 1m, 3m, 5m 거리에서 감지 확인

### 4️⃣ **각도 테스트 (5분)**
- 정면, 측면, 위에서 아래 각도

### 5️⃣ **실시간 검증 (10시간 이상)**
- 웹캠/DroidCam으로 자연스러운 환경 테스트
